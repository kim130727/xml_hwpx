# 전자주보 템플릿 가이드

## 1. 기본 원칙

- 매주 바뀌는 부분만 `{{키}}` 형태의 표식으로 둡니다.
- 제목, 본문, 광고, 표 스타일은 템플릿에서 미리 고정합니다.
- 긴 문장이 들어갈 가능성이 있는 칸은 처음부터 넉넉하게 설계합니다.

## 2. 권장 치환 키

- `{{주보일자}}`
- `{{예배명}}`
- `{{절기}}`
- `{{설교제목}}`
- `{{성경본문}}`
- `{{대표기도}}`
- `{{찬양제목}}`
- `{{헌금찬송}}`
- `{{축도}}`
- `{{광고목록}}`
- `{{교회소식}}`

리스트 값은 자동으로 번호 목록으로 합성됩니다.

## 3. 인쇄 품질을 위한 템플릿 권장값

- 글꼴은 1~2종으로 제한
- 제목, 소제목, 본문, 광고 스타일 분리
- 표 열 너비는 고정
- 광고 영역은 행 높이 자동 확장 여유 확보
- 여백, 머리말, 꼬리말은 템플릿에 고정

## 4. 자동화하기 어려운 항목

- 장평의 미세 조정
- 셀 내부 줄바꿈의 완전 자동 최적화
- 내용 길이에 따라 표를 다시 설계하는 작업

이 부분은 템플릿 설계로 최대한 흡수하고, 생성 후 최종 검수에서 확인하는 것이 안전합니다.
